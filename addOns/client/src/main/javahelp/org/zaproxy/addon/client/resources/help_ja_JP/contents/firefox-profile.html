クライアントサイド統合 - Firefoxプロファイル

このアドオンは 'zap-client-profile' という名前のFirefoxプロファイルを作成し、それをZAPが使用するデフォルトのプロファイルとして設定します。 このプロファイルはすべてのサイトでZAP Firefox拡張機能を有効化します。

別のプロファイルを使用することを選択した場合、ZAPからブラウザを起動する都度、使用するすべてのサイトでZAP拡張機能を手動で承認する必要があります。

これはFirefoxの制限事項です。詳細については、BugzillaのBug 183609を参照してください。

カスタムコンテナ

独自のコンテナでZAP Firefox拡張機能を使用したい場合、そのままでは動作しないことがあります。

'zap-client-profile' が組み込まれているため、ZAP Docker Imagesでは問題なく動作するはずです。 独自のコンテナイメージを作成する場合は、同様にそれを含める必要があるかもしれません。

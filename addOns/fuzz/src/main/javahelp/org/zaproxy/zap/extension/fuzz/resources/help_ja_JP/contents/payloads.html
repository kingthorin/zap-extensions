ペイロードダイアログ

このダイアログでは、リクエストの ファジング 時に使用するペイロードジェネレーターを選択できます。

ペイロードジェネレーターは、ファザーがターゲットアプリケーションに送信する生のデータ値または攻撃を生成します。

既定で提供されているジェネレーターの種類は以下のとおりです：
  • Empty/Null: 選択されたペイロードを複数回生成しますが、メッセージは変更されません。 このペイロードジェネレーターは、例えば ファザー HTTPプロセッサ (スクリプト) などで後から処理される複数のメッセージを送信するのに役立ちます.
  • File (ファイル): 一回限りの攻撃のために、任意のローカルファイルを選択します。
  • File Fuzzers (ファイルファザー): ZAPに登録されているファジングファイル（例：fuzzdbのようなアドオン経由）の任意の組み合わせを選択します。
  • Numberzz (数値): カスタムの増分値を使用して、一連の数値を簡単に生成できます。
  • Regex (正規表現): 正規表現パターンに基づいて攻撃を生成します。
  • Strings (文字列): 生の文字列。手動で入力するか、貼り付けることができます。
  • Script (スクリプト): 必要な任意のペイロードを生成できるカスタムスクリプト
  • Json: 提供されたJSONをファジングすることで攻撃を生成します。
独自のペイロードジェネレータースクリプトを作成できます。これらは、必要なあらゆるペイロードを提供できます。

アドオンは、追加のペイロードジェネレーターを定義することもできます。

「プロセッサ…」ボタンは、 ペイロードプロセッサダイアログ を起動します。これにより、選択したペイロードジェネレーターにのみ適用されるペイロードプロセッサを設定できます。

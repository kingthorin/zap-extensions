パラメーターディガー ダイアログ

このダイアログは、パラメーターディガーを起動します。

コントロール

最初のタブでは、推測機能の選択とともに、URLの選択または変更を行うことができます。
URLが1つ以上のコンテキストに含まれている場合は、その中から1つを選択できます。
また、パラメーターディガーで使用されるスレッド数を変更することもできます。

「URL推測」チェックボックスを選択/チェックすると、「URL推測」タブが表示され、URLパラメーターの推測プロセスを詳細に制御できます。
現時点では、パラメーターディガーは推測攻撃を実行でき、利用可能な推測機能はURLパラメーター推測機能のみであり、これは既定で有効になっています。

「ヘッダー推測」チェックボックスを選択/チェックすると、「ヘッダー推測」タブが表示され、HTTPヘッダーの推測プロセスを制御できます。(まだ利用できません)

「Cookie推測」チェックボックスを選択/チェックすると、「Cookie推測」タブが表示され、Cookieの推測プロセスを制御できます。(まだ利用できません)



URL推測

このタブでは、URL推測手順を設定できます。
「Fcbz キャッシュバスターを追加」を選択/チェックすると、静的なキャッシュバスターがURLに追加/付加されます。 (まだ使用されていません)

「ワードリスト」ドロップダウンでは、ユーザーが自身のワードリストを指定できます。
既定では「定義済み」オプションが選択されており、これは835個のパラメーターで構成される定義済みワードリストを意味します。
「カスタム」オプションを選択すると、独自のワードリストを入力できます。
「両方」オプションを選択すると、定義済みワードリストとユーザーが指定したカスタムワードリストの両方が使用されます。
カスタムワードリストを使用する場合は、単語が改行で区切られていることを確認する必要があります。

現在のリリースでは、このアドオンで使用される定義済みワードリストは、Arjunのスモールワードリストに基づいています。

「チャンク数」フィールドでは、使用されるチャンクの数を制御できます。 1000語を超えるワードリストを使用している場合は、必要に応じてチャンク数を増やすことができます。 そうでない場合は、既定値を使用できます。

「メソッド」リストでは、使用したいメソッドを選択できます。 既定では「GET」が選択されています。 Ctrl + クリック（または Cmd + クリック）を使用するだけで、リストから1つまたは複数のメソッドを選択できます。

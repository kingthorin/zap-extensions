パラメーターディガー

パラメーターディガーは、パラメーターの発見に使用できるツールです。 これにより、攻撃対象領域を拡大し脆弱性発見プロセスを容易にするのに役立つ、隠された、リンクされていない、そして「見つけにくい」パラメーターが特定されます。 これは、与えられたURLをシードとして使用し、ブルートフォースによる推測攻撃を実行してパラメーターを特定します。 これは、主にJames Kettleの研究と実装、すなわちPractical Web Cache PoisoningおよびWeb Cache Entanglementに基づいています。

パラメーターディガーは、パラメーターディガーダイアログを使用して設定および開始できます。

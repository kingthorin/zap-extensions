パッシブスキャナーAPI

APIに以下の操作が追加されます：

アクション

  • clearQueue: パッシブスキャンキューをクリアします
  • disableAllScanners: すべてのパッシブスキャンルールを無効にします
  • disableAllTags: すべてのパッシブスキャンタグを無効にします
  • disableScanners (ids*): パッシブスキャンルールを無効にします
    • ids: スキャンルールIDをカンマで区切ったリスト
  • enableAllScanners: すべてのパッシブスキャンルールを有効にします
  • enableAllTags: すべてのパッシブスキャンタグを有効にします
  • enableScanners (ids*): パッシブスキャンルールを有効にします
    • ids: スキャンルールIDをカンマで区切ったリスト
  • setEnabled (enabled*): パッシブスキャンが有効であるかどうかを設定します。(注: 有効状態は保持されません)
    • enabled: 有効状態 (true または false)
  • setMaxAlertsPerRule (maxAlerts*): パッシブスキャンルールが発行できる最大アラート数を設定します
    • maxAlerts: 最大アラート数
  • setMaxBodySizeInBytes (maxSize*): パッシブスキャナーがスキャンする最大ボディサイズをバイト単位で設定します
    • maxSize: 最大サイズ (バイト単位)、0は未設定を意味します
  • setScanOnlyInScope (onlyInScope*): パッシブスキャンをスコープ内のメッセージに対してのみ実行するかどうかを設定します
    • onlyInScope: スキャン状態 (true または false)
  • setScannerAlertThreshold (id* alertThreshold*): パッシブスキャンルールのアラートしきい値を設定します
    • id: スキャンルールのID
    • alertThreshold: アラートしきい値: OFF、DEFAULT、LOW、MEDIUM、HIGH

ビュー

  • currentTasks: 現在実行中のパッシブスキャンタスクに関する情報 (存在する場合) を表示します
  • maxAlertsPerRule: パッシブスキャンルールが発行すべき最大アラート数を取得します
  • maxBodySizeInBytes: パッシブスキャナーがスキャンする最大ボディサイズをバイト単位で取得します
  • recordsToScan: パッシブスキャナーが今後スキャンする必要のあるレコード数
  • scanOnlyInScope: パッシブスキャンをスコープ内のメッセージに対してのみ実行するかどうかを伝えます
  • scanners: すべてのパッシブスキャンルールを、ID、名前、有効状態、およびアラートしきい値とともにリスト表示します

Statistics Section

The report can also include statistics, per site and global, for example: 
{
    ...
    "site":[
        {
            ...
            "statistics": {
                "site.specific.stat.a": 1,
                "site.specific.stat.b": 2
            }
        }
    ],
    "statistics": {
        "global.stat.a": 1,
        "global.stat.b": 2
    }
}

スクリプトコンソール オプション

コンソール内のスクリプトがディスク上で変更された場合

この設定では、スクリプト コンソールで開いているスクリプトが、例えば別のコードエディターで更新された場合など、ディスク上で変更された場合の既定の動作を設定できます。 選択肢は3つあります：

  • 毎回確認する: スクリプトがディスク上で変更されるたびに確認が表示され、コンソール内のスクリプトを保持するか、変更されたスクリプトに置き換えるかを選択できます。
  • スクリプトを保持: ディスク上で変更された場合でも、常にコンソール内のスクリプトを保持します。
  • スクリプトを置き換える: 常にコンソール内のスクリプトを、変更されたスクリプトに置き換えます。
注: スクリプトに未保存の変更がある場合は、常にどちらのバージョンを保持するかを選択するための確認が表示されます。

フォント

フォント名

スクリプト コンソールで使用されるフォントの名前を設定します。 デフォルトでは、等幅フォントが使用されます。

フォントサイズ

スクリプトコンソールで使用されるフォントのサイズを設定します。 デフォルトのフォントサイズは12です。

コードスタイル

タブサイズ

Tabキーを押したときにコードがインデントされる幅を設定します。 単位は1つのスペース文字の幅です。 デフォルトのタブサイズは4です。

タブ文字を使用

インデントにスペースの代わりにタブ文字 (\t) を使用します。 1つのタブ文字のサイズは、タブサイズの設定によって決定されます。 デフォルトではスペースが使用されます。

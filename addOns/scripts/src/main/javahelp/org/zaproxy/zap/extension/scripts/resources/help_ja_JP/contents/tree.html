スクリプト「ツリー」タブ

スクリプト「ツリー」タブには、現在ロードされているすべてのスクリプトがタイプ別に整理されて表示されます。
また、利用可能なテンプレートも表示されます。これらは直接実行することはできませんが、新しいスクリプトを作成するために使用します。
さらに、新しいスクリプトの追加、ファイルストアからのロード、保存、および削除を行うこともできます。
このタブには、以下の操作を可能にするツールバーが含まれています：

  • 新しいスクリプトの作成
  • ファイルストアからのスクリプトのロード
アクティブスキャナーなどのZAPコンポーネント内で実行されるスクリプトは、右クリックメニューのオプションを通じて有効化および無効化が可能です。
すべてのスクリプトは、右クリックメニューの「スクリプトを削除」オプションを通じてUIから削除できます。

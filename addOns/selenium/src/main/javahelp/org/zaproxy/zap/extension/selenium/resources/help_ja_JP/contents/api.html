Selenium API

'selenium' APIを使用すると、必要なWebDriver群およびバイナリへのパスを設定および表示できます。

APIアクション/ビュー

名前 タイプ パラメーター 説明
getBrowserArguments ビュー browser* ブラウザ引数を取得します。
optionChromeBinaryPath ビュー 現在のChromeバイナリへのパスを返します。
optionChromeDriverPath ビュー 現在のChromeDriverへのパスを返します。
optionFirefoxBinaryPath ビュー 現在のFirefoxバイナリへのパスを返します。
optionFirefoxDriverPath ビュー 現在のFirefoxドライバー (geckodriver) へのパスを返します。
addBrowserArgument アクション browser* argument* enabled ブラウザ引数を追加します。
launchBrowser アクション browser* ZAPをプロキシとして使用してブラウザを起動し、手動で使用します。
removeBrowserArgument アクション browser* argument* ブラウザ引数を削除します。
setBrowserArgumentEnabled アクション browser* argument* enabled* ブラウザ引数を有効にするかどうかを設定します。
setOptionChromeBinaryPath アクション 文字列* 現在のChromeバイナリへのパスを設定
setOptionChromeDriverPath アクション 文字列* 現在のChromeDriverへのパスを設定
setOptionFirefoxBinaryPath アクション 文字列* 現在のFirefoxバイナリへのパスを設定
setOptionFirefoxDriverPath アクション 文字列* 現在のFirefoxドライバー (geckodriver) へのパスを設定
アスタリスク (*) 付きのパラメータは必須です。

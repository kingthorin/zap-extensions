WebSocketタブ

WebSocketタブには、すべてのWebSocket接続メッセージが表示されます。 ZAPがアクティブな間に、例えば Mozilla's BrowserQuest にアクセスして、WebSocketの動作を確認できます。

フィルタ

チャネルセレクター

表示するメッセージを、特定の単一WebSocketチャネルに限定したり、すべてのチャネルを表示したりできます。
特定のチャネルを選択した場合、 ハンドシェイクを表示 ボタン () が有効になります。このボタンをクリックすると、対応するHTTPハンドシェイクが リクエスト/レスポンス タブに表示されます。

フィルタ ダイアログ

フィルタダイアログ () では、さらに表示されるメッセージを制御できます。

ブレークポイント

You can set custom breakpoints using the カスタムブレークポイントを追加 ボタン ().
または、 右クリックメニューを使用することもできます.

右クリック メニュー

WebSocketメッセージを右クリックすると、以下の操作を可能にするメニューが表示されます：

WebSocketから除外

WebSocket接続は セッションプロパティに除外URLを追加することで、WebSocketタブから除外できます。 この項目をクリックすると、URLが事前に入力された状態でセッションプロパティが表示されます。 WebSocket接続を除外してもその接続は閉じられず、それ以降の処理なしで転送されます。

ブレーク...

これにより、カスタムブレークポイントを設定できる ブレークポイントの追加ダイアログ が表示されます。
